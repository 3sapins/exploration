# Pack d’images — Charte historique

Ce dossier contient des **placeholders SVG** pour chaque scène du jeu. Remplacez-les par des **images historiques** libres de droits (JPEG/PNG) en conservant **les mêmes noms de fichiers** pour que l’association automatique fonctionne.

## Convention
- Parcours **Cortés (espagnols)** → fichiers `conquistador_*.svg` (gravures, cartes, iconographie européenne).
- Parcours **Moctezuma (mexica)** → fichiers `codex_*.svg` (pages de codex : Florentin, Lienzo de Tlaxcala, etc.).

Vous pouvez remplacer un `.svg` par un `.jpg` ou `.png` sans rien changer dans le code (gardez le **même nom** et l’extension correspondante, pensez à mettre à jour `imageMap` si vous changez l’extension).

## Où trouver des images libres
- **Codex Florentin (Livre XII)** — numérisations en domaine public (Biblioteca Medicea Laurenziana / Wikimedia Commons).
- **Lienzo de Tlaxcala** — fac-similés et extraits disponibles en dépôt public.
- **Bibliothèque du Congrès** / **Gallica BnF** / **Wikimedia Commons** — images de conquistadors, gravures, cartes du XVIe siècle.

## Accessibilité
- Ajoutez des attributs `alt` descriptifs aux images si vous changez l’implémentation.
